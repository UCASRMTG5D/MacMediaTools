# MacMediaTools

MacMediaTools 是一个 macOS 媒体处理工具集，面向日常的视频、音频和图片整理工作。它把常用的小工具放进同一个桌面应用里，方便直接处理文件，不用在多个软件之间来回切换。

## 主要功能

- 宽高调整
- 视频片段整合
- 音视频处理
- 批量截图
- 重复照片检测
- 重复视频检测
- 重复媒体检测
- 文件复制工具

## 关键特性

- 批量截图支持时间范围选择、逐帧预览、质量检查、内容去重和 ZIP 导出。
- 音视频处理支持导入视频和音频、调整偏移、预览时间轴并导出结果。
- 重复媒体检测支持按照片、视频和混合媒体分类扫描。
- 宽高调整、拼接和复制都集成在同一个窗口里。
- 构建完成后会自动把生成的 `.app` 复制到 `Applications`，方便直接使用，并覆盖旧版。

## 环境要求

- macOS 13 或更高版本
- Xcode 15 或更高版本

## 构建与运行

1. 打开 `MacMediaTools.xcodeproj`
2. 选择 `MacMediaTools` Scheme
3. 编译并运行

如果本机允许写入系统应用程序目录，构建完成后应用会自动出现在 `/Applications/MacMediaTools.app`。  
如果系统不允许直接写入，也会尝试安装到当前用户的 `~/Applications`。

## 使用说明

- 启动后在左侧选择功能。
- 每个工具都支持选择输入文件、设置参数并执行处理。
- 处理完成后通常可以直接在 Finder 中打开结果目录或结果文件。

## 备注

- 项目会持续围绕稳定性、交互手感和工具可用性进行优化。
