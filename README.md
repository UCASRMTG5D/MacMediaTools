# MacMediaTools（macOS / Apple Silicon 优先）

一个**纯本地**的 macOS SwiftUI 小工具集合（GUI 应用），面向日常视频处理与重复文件排查场景。

> 说明：项目主要在 Apple Silicon（M 系列）环境下开发/测试；理论上也可在 Intel Mac 上编译运行（以实际环境为准）。

## 功能一览

打开应用后，左侧先选择功能：

- **视频尺寸修改**：将视频导出为指定分辨率，解决“分辨率不匹配导致画面看起来被拉伸”的问题
- **视频尺寸裁剪**：预览视频并拖动黄色裁剪框，导出裁剪后的新视频
- **多个视频拼接**：批量选择视频、列表内拖动排序，然后拼接导出
- **重复照片检测**：递归扫描文件夹（含子文件夹），按 **SHA256** 内容哈希分组输出重复文件
- **重复视频检测**：递归扫描文件夹（含子文件夹），仅按 **时长 / 文件大小 / 分辨率** 分组；**不做内容检测**（界面有提示）

## 环境要求

- macOS **13.0+**
- Xcode（建议使用与 macOS 版本匹配的较新版本）

## 编译 / 运行

1. 用 Xcode 打开项目根目录下的 `MacMediaTools.xcodeproj`
2. 选择运行目标 `MacMediaTools`（My Mac）
3. `Run`（⌘R）

依赖说明：

- 不依赖 Homebrew / ffmpeg / CocoaPods
- 视频处理基于系统框架 **AVFoundation / AVKit**
- 图片重复检测使用 **CryptoKit（SHA256）**

## 详细使用说明

### 1) 视频尺寸修改

入口：左侧 **视频尺寸修改**

流程：

1. 点击「选择视频…」选取一个视频
2. 程序会自动读取并显示原始“显示宽高”（已考虑视频旋转信息）
3. 输入「新宽度 / 新高度」
4. 选择「输出目录 / 输出文件名」
5. 点击「开始处理」，导出新视频（MP4：H.264 + AAC）

默认值（与当前代码一致）：

- 输出目录：默认为**原视频同目录**
- 输出文件名：默认为 `原文件名_1.mp4`
- 缩放策略：
  - **拉伸到目标分辨率（会改变比例）**（默认）
  - **保持比例加黑边**（Aspect Fit）

### 2) 视频尺寸裁剪

入口：左侧 **视频尺寸裁剪**

流程：

1. 点击「选择视频…」
2. 程序会显示原始“显示宽高”，并在预览上显示**黄色裁剪框**
3. 拖动裁剪框四角或整体移动，确定要裁剪掉的边缘
4. 选择「输出目录 / 输出文件名」
5. 点击「开始裁剪」，导出裁剪后的视频（MP4：H.264 + AAC）

默认值：

- 输出目录：默认为**原视频同目录**
- 输出文件名：默认为 `原文件名_1_crop.mp4`

### 3) 多个视频拼接

入口：左侧 **多个视频拼接**

流程：

1. 点击「批量选择视频…」选择多个视频（可多选）
2. 在列表中**拖动**调整拼接顺序（也支持删除单个、清空）
3. 选择「输出目录 / 输出文件名」
4. 点击「开始拼接」，导出拼接后的视频（MP4：H.264 + AAC）

默认值：

- 输出目录：默认为**第 1 个视频同目录**
- 输出文件名：默认为 `merged_1.mp4`

当前限制（MVP）：

- 目前要求所有输入视频的“显示分辨率”一致；不一致会提示并阻止导出（后续可扩展为自动缩放到统一分辨率）。

### 4) 重复照片检测（递归）

入口：左侧 **重复照片检测**

检测规则：

- 递归扫描所选文件夹及其子文件夹
- 仅扫描常见图片扩展名：jpg/jpeg/png/heic/heif/tif/tiff/bmp/gif/webp
- 对每个文件做 **SHA256**（分块读取，避免一次性读入内存）
- **SHA256 完全一致** ⇒ 判定为“完全相同文件”，并分组输出

结果展示：

- 以“重复组”的形式展示，每组可展开查看文件列表
- 每个文件支持「在 Finder 中显示」

### 5) 重复视频检测（递归，仅基本信息）

入口：左侧 **重复视频检测**

界面提示（与代码一致）：

> 只检测视频文件基本信息（时长 / 文件大小 / 分辨率），暂不支持内容检测～

检测规则：

- 递归扫描所选文件夹及其子文件夹
- 仅扫描常见视频扩展名：mp4/mov/m4v/avi/mkv
- 以 `(时长ms, 文件大小B, 显示分辨率wxh)` 作为分组 key
- key 完全一致 ⇒ 归为同一组“基本信息重复”

## 隐私与安全

- 所有处理与扫描都在本机完成
- 不上传文件、不联网、不收集用户数据（项目代码内未包含网络请求逻辑）

## 项目结构（简要）

- `MacMediaTools/RootView.swift`：左侧功能选择 + 详情页切换
- `MacMediaTools/Features/*`：各功能界面（Resize/Crop/Concat/Duplicate）
- `MacMediaTools/Services/VideoToolkit.swift`：视频信息读取与导出（resize/crop/concat）
- `MacMediaTools/Services/FolderScanner.swift`：递归扫描文件夹
- `MacMediaTools/Services/FileHasher.swift`：SHA256 计算

## 已知限制（MVP）

- **视频裁剪/尺寸修改**：对于带有复杂旋转元数据的视频（例如竖屏 + rotation），裁剪框与实际裁剪结果可能存在偏差；后续可进一步完善坐标映射。
- **视频拼接**：当前要求输入视频显示分辨率一致；不一致会提示并阻止导出。
- **重复视频检测**：仅检测基本信息（时长/大小/分辨率），不做内容指纹对比。
